# Draft Canvas — はじめてガイド

> Mac で AI 画像を生成・編集したい人向けの導入ガイドです。

---

## Draft Canvas とは？

Draft Canvas は、Mac で動く AI 画像生成・編集アプリです。

テキストを打ち込むだけで画像が生成されます。生成した画像はそのままアプリ内で編集でき、背景の除去・高解像度化・SVG 変換まで一気通貫でこなせます。サブスクリプションなし、¥2,000 の買い切り。

## はじめ方

### ステップ 1 — Codex CLI をインストールする

Draft Canvas は Codex CLI を通じて AI に画像を生成させます。先にインストールしてください。

```bash
npm install -g @openai/codex
```

インストール後、一度 `codex` を実行して OpenAI / ChatGPT アカウントでログインしておきます。

### ステップ 2 — Draft Canvas をダウンロードする

[無料トライアルをダウンロード](https://draftcanvas.lemonsqueezy.com/checkout/buy/2330d4a5-c212-4248-8053-07993e8522e4) から `.dmg` を入手し、アプリをアプリケーションフォルダにコピーします。

**トライアル:** 14日間、クレジットカード不要で全機能が使えます。

### ステップ 3 — アプリを起動する

初回起動時に通知の許可を求められます。「許可」しておくと、生成完了時に通知が届きます。

## 基本的な使い方

### 画像を生成する

1. 左サイドバーの **「+」** ボタンで新しいプロジェクトを作成
2. 画面下のテキストエリアにプロンプトを入力（例: `白い猫がコーヒーを飲んでいる、水彩画風`）
3. アスペクト比・生成枚数を必要に応じて変更
4. **「生成」** ボタンをクリック

生成完了するとキャンバスに画像カードが並びます。

**ヒント — プロンプト強化機能:**  
プロンプト入力後に ✨ ボタンを押すと、AI が自動的に詳細な説明を補完します。クオリティが上がりやすくなります。

### 画像を編集する

キャンバス上の画像カードを **クリックして選択** すると、キャンバス左側に丸いアクションボタンが縦に並んで表示されます。

| ボタン | できること |
|-------|----------|
| 再編集 | この画像を元に AI が新しい画像を生成 |
| マスク編集 | ブラシで範囲を指定して、その部分を AI に修正 or 除去させる |
| 背景を除去 | Apple の AI で背景をオフライン除去 |
| 素材として分離 | 画像からマテリアルを抽出して新しいアイテムとして保存 |
| 高解像度化 | AI でアップスケール |
| ベクター化 | ラスター画像を SVG に変換 |
| 複製 | 同じ画像を複製してキャンバスに追加 |
| エクスポート | 保存形式を選んで書き出す |
| 削除 | キャンバスから削除 |

### マスク編集（インペイント）

特定の部分だけを変えたいときに使います。

1. 画像をクリックして選択 → 左パネルの「マスク編集」をクリック
2. ブラシで変えたい箇所を塗る
3. マスクエディタ内でモードを選択して生成:
   - **編集**: プロンプトに指示を入力して「生成」をクリック（塗った範囲を AI が書き換える）
   - **除去**: 「生成」をクリックするだけ（塗った範囲のオブジェクトを背景で自然に埋める）

### 画像をエクスポートする

1. 画像をクリックして選択 → 左パネルの「エクスポート」をクリック
2. 形式・品質・リサイズを設定
3. 「書き出す」をクリック

初回はフォルダ選択が表示されます。以後は同じフォルダに保存されます。

**エクスポート形式:**

| 形式 | 用途 |
|------|------|
| PNG | 高品質。Web・印刷用 |
| JPEG | ファイルサイズを小さくしたい場合 |
| SVG | ベクター化済みの場合のみ。拡大しても劣化しない |

## 主な機能

### AI 画像生成

| 設定 | 説明 |
|------|------|
| プロンプト | 日本語・英語どちらでも OK |
| アスペクト比 | 正方形 / ポートレート / ストーリー / 横長 / ワイドスクリーン / 自動 |
| 生成枚数 | 1〜8枚 |
| 並列数 | 何枚同時に生成するか（レート制限検知時は自動縮減） |
| モデル | GPT-5.5（賢い・高コスト）〜 gpt-5.2（軽量・低コスト）から選択 |

### プロジェクト管理

- 生成した画像はプロジェクトごとにキャンバスで管理
- 複数プロジェクトをサイドバーで切り替え
- タグを付けて検索・整理できる
- **フィルタリングプロジェクト**: 検索条件を保存して横断ビューを作れる

### 使用量の確認

TopBar に 5時間・週次の使用量ピルが常時表示されます。各ピルには使用量・残量バー・「あと何日何時間」のリセット目安が表示されます。アカウントアイコンをクリックすると、プラン名・アカウント種別などの詳細を確認できます。

## よくある質問

**Q. 生成に失敗する / エラーが出る**  
A. Codex にログインしているか確認してください（`codex` をターミナルで実行してログイン状態を確認）。

**Q. 画像の品質やモデルの種類は Draft Canvas のバージョンで決まる？**  
A. いいえ。画像生成の実行は Codex に委任しているため、**利用可能なモデル・画像品質・解像度は Codex のバージョンに依存**します。Draft Canvas 自体のアップデートとは独立しており、Codex を更新するだけで使えるモデルや生成品質が変わることがあります。Draft Canvas 側が制御できるのはプロンプト・Reasoning Effort・モデル選択のみです。

**Q. 背景除去はオフラインで使える？**  
A. はい。背景除去は Apple Vision Framework を使って Mac 上で完結します。インターネット不要です。

**Q. 生成した画像はどこに保存される？**  
A. `~/Library/Application Support/Draft Canvas/` 内に自動保存されます。エクスポートで任意のフォルダに書き出せます。

**Q. 1ライセンスで何台使える？**  
A. 最大 2台（例: 自宅 Mac ＋ 仕事 Mac）まで使えます。

**Q. 返金できる？**  
A. 購入から 30日以内であれば全額返金に対応しています。
