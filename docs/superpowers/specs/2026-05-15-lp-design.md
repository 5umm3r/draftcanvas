# Draft Canvas LP 設計仕様

## 目的

Lemon Squeezy / Stripe 審査通過 + ユーザーへのダウンロード導線確保。  
`spade3.com` に FTP アップロードする静的 HTML 一式を作成する。

## 確定済み設計判断

| 項目 | 決定 |
|---|---|
| 言語 | 日英両対応 (単一ページ、日本語メイン・英語サブ表記) |
| ホスティング | 静的 HTML → FTP アップロード (spade3.com) |
| デザインスタイル | ダーク × グラデーション (紫→青) |
| Hero レイアウト | 左テキスト + 右スクリーンショット |
| キャッチコピー方向 | 差別化訴求 (買い切り・サブスクなし) |

## ファイル構成

```
spade3.com/
├── index.html          # メイン LP
├── privacy.html        # プライバシーポリシー (審査必須)
├── terms.html          # 利用規約 (審査必須)
└── assets/
    ├── screenshot.png  # アプリスクリーンショット (Hero 右側)
    └── icon.png        # アプリアイコン
```

## 技術仕様

- 単一 HTML ファイル (外部 CSS/JS ファイルなし → FTP が楽)
- モバイルレスポンシブ対応 (ただし PC 優先)
- font: `-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`
- カラー: bg `#080810`, text `#e4e4e7`, accent `#a78bfa`→`#60a5fa`
- グラデーション: `linear-gradient(135deg, #7c3aed, #2563eb)`
- 外部フォント・アイコンライブラリは使用しない

## 実装環境要件

- **Codex CLI インストール済み必須** (ファイル生成・編集に使用)

## 注意事項

- 外部 CDN (Google Fonts, Bootstrap 等) を使わない → FTP 環境でネット不要、全ローカルファイルで完結
- `assets/` フォルダのファイルは FTP アップロード前にユーザーが手動配置
- LS の checkout URL はテストモードと本番モードで共通の UUID (2330d4a5-...) を使用
- Privacy Policy / Terms はテンプレートベースで作成するが、法的拘束力の確認はユーザー責任
