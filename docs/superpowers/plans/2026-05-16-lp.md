# Draft Canvas LP 実装プラン

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Draft Canvas の LP を静的 HTML 3ファイルで作成し、spade3.com に FTP アップロードできる状態にする

**Architecture:** 外部依存ゼロの静的 HTML。CSS はインライン style タグ内に記述。JavaScript は最小限 (言語切替のみ)。スクリーンショット等のアセットは `lp/assets/` に配置し FTP で一括アップロード。

**Tech Stack:** HTML5, CSS3 (インライン), Vanilla JS (言語切替のみ), Codex CLI

**スペック:** `docs/superpowers/specs/2026-05-15-lp-design.md`

---

## ファイル構成

```
lp/
├── index.html      # メイン LP (Hero / Features / Pricing / FAQ / Footer)

```
