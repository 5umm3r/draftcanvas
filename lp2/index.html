FAQ

ChatGPT のアカウントが必要ですか？

画像の生成・編集機能には ChatGPT の有料プラン（Plus・Pro・Team 等）が必要です。背景除去・ベクター化・オブジェクト切り出しはオフラインで動作し、ChatGPT 不要です。

Codex CLI とは何ですか？インストール方法は？

OpenAI が提供するターミナル用 AI ツールです。Draft Canvas はこれを経由して ChatGPT に接続します。npm install -g @openai/codex でインストールし、codex コマンドでログインが必要です。詳しくは Codex CLI 公式リポジトリをご参照ください。
