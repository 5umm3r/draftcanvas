FAQ

FAQ

Do I need a ChatGPT account?

Image generation and editing features require a ChatGPT paid plan (Plus, Pro, Team, etc.). Background removal, vectorization, and object extraction work offline and do not require ChatGPT.

What is Codex CLI and how do I install it?

Codex CLI is a terminal-based AI tool from OpenAI. Draft Canvas uses it to connect to ChatGPT. Install it with npm install -g @openai/codex, then run codex to log in. See the Codex CLI repository for details.

Are there extra fees for image generation?

Draft Canvas costs ¥1,980 (one-time). Image generation uses your existing ChatGPT paid plan — no additional per-image charges apply.

Is there a subscription?

Will I be charged automatically after the trial?

No. The 14-day trial requires no credit card and does not auto-renew.

Does it work on Intel Macs?

Yes. Draft Canvas supports both Apple Silicon and Intel Macs running macOS 14 Sonoma or later.
